Two - The World's Opinion

I designed and created a social networking/polling app that allows users to create, vote on, and see detailed results for simple questions from around the world. The app uses a card swiping interface to allow users to answer questions as quickly as possible. To build this project, I used Google Firestore primarily for the backend, along with the Firebase Realtime Database for storing results, Firebase Storage, and Firebase Cloud Functions to connect all of the pieces together. In addition to Firebase, Two also uses Algolia for user search, and Branch.io for dynamic links.
